# UNISOLE AI Exploration Lab

Starter structure for an interactive AI learning project.

## Folders

- `assets/`: images, icons, animations, and CSS.
- `data/`: datasets and example files.
- `utils/`: reusable helpers, prompts, styling, and chart utilities.
- `pages/`: numbered lesson pages.
- `notebooks/`: exploratory notebooks.

## Run

Update `app.py` with your preferred app framework entry point, then install:

```bash
pip install -r requirements.txt
python app.py
```
